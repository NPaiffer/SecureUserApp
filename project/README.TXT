Estrutura do Projeto:
questao1.py: Contém a função para criar e validar logins e senhas.
questao2.py: Implementa o cadastro, login e atualização de usuários.
questao3.py: Adiciona a funcionalidade de atualização de login e senha.
questao4.py: Implementa a busca do maior número primo até um valor N.
questao5.py: Protege as senhas dos usuários utilizando criptografia.
questao6.py: Atualiza o login com a senha protegida.
questao7.py: Persiste o banco de dados de usuários em um arquivo JSON.
questao8.py: Atribui um ID único a cada usuário.
questao9.py: Conecta-se à API de Pokémon para associar um Pokémon ao ID de cada usuário.
questao10.py: Atualiza o banco de dados de usuários após a associação do Pokémon.

Funcionalidades Principais:
Cadastro de usuários com validação de login e senha
Login de usuários com verificação de autenticidade
Atualização de informações de login e senha
Associação de um Pokémon ao ID de cada usuário
